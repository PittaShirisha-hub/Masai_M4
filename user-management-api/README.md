# User Management API

Backend REST API built using Node.js, Express, and Supabase.

## Features
- User CRUD operations
- Input validation
- Password hashing
- Supabase PostgreSQL database

## Tech Stack
- Node.js
- Express.js
- Supabase
- PostgreSQL

## API Endpoints
POST /api/users  
GET /api/users  
GET /api/users/:id  
PUT /api/users/:id  
DELETE /api/users/:id  

## Run Project
npm install  
npm run dev
